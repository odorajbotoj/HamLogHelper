# HamLogHelper

HamLogHelper

by BG4QBF

## 页面说明

+ 左侧
  + 上半部分为已经记录的日志
  + 下半部分为正在输入的日志。“锁定”可以保持当前信息提交时不改变
+ 右侧
  + 上半部分为地图，可以查看已通联的友台 QTH （基于天地图）
  + 下半部分为搜索框，可以搜索地点，点击地点会将信息填入“对方台址”一栏

## 使用说明

+ 在可执行文件同级目录下创建 `tianditu-key.txt` ，填入天地图 api-key
+ 访问浏览器 `localhost:5973` 打开前端页面
+ 每打开一个新的页面，都会新生成一个 csv 文件用于记录。但通联编号是连续的，仅在关闭后台后清除

## 数据说明

+ 标准 csv ，逗号分隔，半角引号 `"` 转义为双半角引号 `""` ，所有字段都有引号包裹
+ 每一行分别是以下数据
  1. 位号
  2. 呼号
  3. 日期时间 ( UTC )
  4. 频率
  5. 模式
  6. 信号
  7. 对方设备
  8. 对方功率
  9. 对方天线
  10. 对方台址
  11. 己方设备
  12. 己方功率
  13. 己方天线
  14. 己方台址
  15. 备注
